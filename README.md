Odin Recipes - The Odin Project Foundations

This is my recipes project for The Odin Project Foundations path. 

In this project, we were asked to make a simple website using only HTML. It includes three of my favorite recipes with images for each dish. The main index page and all three recipe pages are linked together using relative file paths to demonstrate basic HTML structure and navigation.

Featured Recipes:
1. Masala Dosa
2. Mango Lassi
3. Chole Bhature
